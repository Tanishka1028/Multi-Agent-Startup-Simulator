# Multi-Agent Startup Simulator

An AI-powered business simulation platform built using Python, Streamlit, and Groq API that simulates how different departments inside a startup collaborate to evaluate and develop a business idea.

The application uses multiple AI agents acting as:

* CEO Agent
* Finance Agent
* Marketing Agent
* SWOT Analysis Agent

Users can enter any startup idea and instantly receive:

* Startup vision and mission
* Business model suggestions
* Financial analysis
* Marketing strategies
* SWOT analysis
* Revenue forecasts

# Installation

## 1. Clone Repository

```bash
git clone <your-repository-link>
```

---

## 2. Create Virtual Environment

### Windows

```bash
python -m venv venv
```

Activate environment:

```bash
venv\Scripts\activate
```

---

## 3. Install Dependencies

```bash
pip install -r requirements.txt
```

---

# Environment Variables

Create a `.env` file:

```env
GROQ_API_KEY=your_api_key_here
```

---

# Run the Project

```bash
python -m streamlit run main.py
```
